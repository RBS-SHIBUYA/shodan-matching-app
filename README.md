商談マッチングアプリ(試作)

展示会・商談会における出展者とバイヤーの商談を管理する業務システムの試作(プロトタイプ)です。
Claude Artifact 上で動作する単一の React ファイルとして、段階的に機能を追加しながら開発しています。

開発目的

展示会・商談会運営における以下の業務を一元管理し、事務局の作業負担を削減することを目指しています。


出展者・バイヤー・商品情報の管理
進捗管理とリマインド(メール・SMS)
面談希望の受付とマッチング
タイムスケジュールの自動作成
対応状況・送信履歴の確認


技術構成


React(単一ファイル・Reactフックのみ使用)
Tailwind CSS(ユーティリティクラス)
lucide-react(アイコン)
外部バックエンドなし。サンプルデータと画面内の状態のみで動作する試作版です


ファイル構成

現時点では単一ファイルです。

shodan-matching-app-stage5.jsx

Claude.ai の Artifact 機能でこのファイルの中身をそのまま貼り付けることで、ブラウザ上で動作を確認できます。

開発段階(現在: 第5段階まで完了)

段階内容第1段階進捗管理・リマインド管理(メール)第2段階CSV一括アップロード・会社情報・商品情報・商品画像第3段階面談希望管理・マッチング候補一覧(自動スコアリング)第4段階タイムスケジュール自動作成・確認状況の管理第5段階SMSリマインド管理(送信候補抽出・承認フロー・送信履歴)第6段階以降送信履歴の統合・各種設定 など(今後の予定)

試作段階での制約


実際のメール配信・SMS配信サービスへは接続していません(送信は模擬的に成功・失敗を表示します)
データはブラウザを閉じると保持されない場合があります(Claude.ai の Artifact 環境の永続ストレージ機能を利用している場合は保持されます)
サンプルデータ(出展者10社・バイヤー5社・商品20件 等)を初期状態として含みます
